## About Thunder Executor

Thunder Executor is a Windows-based Lua injector made for Roblox. It gives users a simple route for loading and running custom scripts inside the Roblox client, backed by a script hub that includes more than 500 ready-made scripts from different game categories. The app is intentionally lean and fast, with minimal overhead for users who want a focused execution tool rather than a bloated suite.

It also ships with an auto-update system that helps keep compatibility aligned with the newest Roblox client releases, plus a persistent SQLite-powered queue for controlling script execution order. A multi-language interface makes it usable across regions, and the small install size under 30 MB helps it stay lightweight on most Windows PCs.
